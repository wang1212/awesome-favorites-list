Programming language.

    • 《C 程序设计语言》[美] Dennis M. Ritchie, Brian W. Kernighan

    • 《JavaScript 面向对象编程指南》[加拿大] Stoyan Stefanov 斯托扬
    • 《高性能 JavaScript》 [美] 尼古拉斯·泽卡斯
    • 《Shell scripting with Node.js》 Dr. Axel Rauschmayer - 免费书籍。

    • The best free Python resources
    • 《Python 101》 (美) Mike Driscoll - 讲述 Python 3 初级编程的书籍。

  • Ruby

    • 《Ruby 元编程》[意] Paolo Perrotta
    • 《Ruby 基础教程》[日] 高桥征义 后藤裕藏
    • 《The Book Of Ruby》 (英) Huw Collingbourne - 关于 Ruby 编程的开源书籍，涉及知识概念很全面，开源书籍。 👉 简体中文翻译
    • 《Ruby For Beginners》 - Ruby 初学者向导，开源书籍。

  • Bash

    • 《Advanced Bash-Scripting Guide》 - 高级 Bash 脚本编程指南，开源书籍。

    • Physically-Based Rendering, And You Can Too!

  • Introduction to Computer Graphics - 计算机图形学概论，可在线免费阅读。
