在线资源​

Online Resources.

设计原则​

Design Principles.

  • Design Systems Database - 一流的设计系统，其组件和基础参考来自顶级科技公司和领先的 UI 团队。
  • lawsofux - 构建用户界面时可以参考的最佳实践。

设计工具​

Design Tools.

  • Dribbble - 设计风格参考。
  • lapa - 主流的落地页设计参考。
  • refero - 最大的网页和 iOS UI/UX 设计灵感库。
  • Untitled UI - 全球最大的 Figma UI 工具包和 React 组件库。
  • Free Resources - 免费的设计资源。
  • 设计师网址导航
  • 设计导航
  • Boss 设计 - 收集国内外设计素材资源的网站。
  • UI 中国用户体验设计平台
  • 花瓣网 - 设计素材网站。
  • Zoomstock - 搜索引擎，可索引来自免费顶级图片网站的数百万张照片和插图。
  • namelix - 输入关键字，利用人工智能生成品牌名称。
  • hatchful - logo 设计。
  • logofury - logo 设计。
  • Font Pair - 排版工具。
  • CG Society
  • Code My UI - Web UI 设计相关的代码片段。

配色方案​

Coloring scheme.

  • Colormind - 利用深度学习生成配色方案。
  • ColorMagic - AI 调色板，通过关键字生成调色板。
  • Open Color
  • Color Hunt
  • coolors
  • Sorted CSS Colors
  • Grabient - 渐变色。
  • material design palette - Google Material Design 设计工具，提供官方图标（Material Icons）下载，非官方。

图像工具​

Image tool.

占位图像服务​

Placeholder image service.

  • Placeholder.com - 图片、灰度图、文本。
  • Fake images please? - 灰度图、文本，免费开源。
  • Dynamic Dummy Image Generator - 灰度图、文本。
  • Lorem Picsum - 图片。
  • PlaceIMG - 图片。
  • Ipsum Image - 灰度图、文本。

动画​

  • LottieFiles - 动画文件，适用于 Web 与移动客户端开发。

其它​

  • BGJar - 在线 SVG 背景图生成。
  • Trianglify.io - 生成三角网 SVG 背景图。
  • Hero Patterns - 一些 SVG 背景图。
  • Get Waves - 生成波浪 SVG 背景图。
  • Cool Backgrounds - 利用渐变，生成 PNG 背景图。

素材资源​

Material Resources.

图片素材​

Picture material.

  • Pexels - 免费的摄影素材分享。
  • Unsplash - 免费的图片素材。
  • Pikwizard - 免版税的图片、视频素材。
  • Vecteezy - 图片素材。
  • MixKit - 图片素材。

插画素材​

Illustration material.

  • unDraw - 插画素材，扁平风格，免费下载、开源。
  • Open Doodles - 插画素材，免费下载、开源。
  • Open source illustrations kit - 插画素材，免费下载、开源。
  • Lukasz Adam Free Illustrations - 插画素材，免费下载、开源。
  • Ouch - 插画素材，免费下载，Icons8 提供。
  • DrawKit- 插画素材，矢量图标，免费下载。
  • Awesome illustrations - 插画素材，免费下载。
  • Delesign- 插画素材，矢量图标，免费下载。
  • Gallery.Manypixels - 插画素材，免费下载。

字体图标​

Fonts and Icons.

  • 字体工具

    • fontfamily.io - 查看字体在各个平台的兼容性。
    • Modern Font Stacks - 现代字体堆栈。
    • uifonts - 实时预览字体效果。
    • Everything Fonts - 字体处理。
    • cn-font-split - 字体分割工具。

  • 字体资源

    • Google Fonts - 免费开源的字体集合。
    • Google Noto Fonts - 免费可供所有人使用的，支持全球大部分语言的字体，Google 发布。
    • Fontshare - 可免费商用的字体集合。
    • Fontsource - 开源字体资源，提供 CDN、npm 包。

  • 图标工具

    • Free Favicon Maker - SVG 图标制作。
    • IconKitchen - App 图标设计工具。
    • Icons8 - 免费图标，设计工具。
    • iconbundler - 将 SVG 图标导出为 React、Vue、Svelte 组件。

  • 图标素材

    • icon hunt - 免费、开源的图标搜索引擎。
    • iconoir - 图标库，免费开源。
    • Iconduck - 图标与插画素材，免费开源。
    • tabler-icons - 免费开源。
    • Iconbuddy - 免费开源。
    • Glyphs - 图标库，开源。
    • iconsax - 图标库，可免费商用。
    • Simple Icons - 品牌 logo 图标，仅支持 SVG，免费。
    • Forge Icons - 图标库，仅支持 SVG，可免费商用。
    • Feather - 简单漂亮的图标库，仅支持 SVG，开源。
    • flag-icons - 世界各国国旗图标，支持 SVG 与 CSS，开源。
    • Flat Icon - 图标市场。
    • Bootstrap Icons - 开源图标库，仅支持 SVG，Twitter 公司发布。
    • Remix Icon - 中性风格的开源图标库，支持 SVG 与 Web font。
    • IonIcons - 精美的开源图标库，支持 SVG 与 Web font，ionic 团队制作。
    • Material Icons - Material Design 风格的开源图标库，支持 SVG 与 Web font，Google 官方发布。
    • Material Design Icons - Material Design 风格的开源图标库，支持 SVG 与 Web font非官方，社区支持。
    • Octicons - GitHub 开源图标库，GitHub 官方发布。
    • Teenyicons - 开源图标库，仅支持 SVG。
    • Fabric Icons - 图标库，仅支持 Web font，Microsoft 官方发布。
    • Spectrum Icons - Adobe Spectrum 设计系统图标，Adobe 官方发布。
    • Font Awesome - 丰富的图标库，提供免费版。
    • IconPark - 开源免费的图标库，字节跳动公司发布。
