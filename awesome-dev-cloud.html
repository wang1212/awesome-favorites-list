开发技术​

Development Technology.

  • Docker - 轻量的类似虚拟机的容器技术。

    • HEALTHCHECK - 容器健康检查的一些示例程序，官方维护。
    • Awesome-docker
    • CLI Tools
      • ctop - 类似于 top 命令，容器性能指标监控工具。
      • lazydocker - Docker 终端 UI 管理工具。

  • Kubernetes - k8s，分布式容器编排工具，自动化部署、管理，CNCF 毕业。

    • kaniko - 在容器内构建镜像。
    • Articles
      • Kubernetes tools and resources
      • Graceful shutdown and zero downtime deployments in Kubernetes
    • k3s - 轻量级，适用于边缘计算等场景的 Kubernetes 发行版。

  • OpenTelemetry - 由 OpenTracing 和 OpenCensus 项目合并而成，性能数据、日志数据追踪、收集，CNCF 孵化。

    • Jaeger - 分布式追踪系统，Uber 开源，CNCF 毕业。

  • Grafana - 监控数据可视化、图表、仪表盘面板。

  • CI/CD

    • Drone - CI 平台。
    • Earthly - CI/CD 管道框架。
