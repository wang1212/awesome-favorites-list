服务器​

Containers that run server-side programs and provide static resource services.

  • Nginx - 轻量级、高性能、高并发的 Web 服务器，主要作为负载均衡、反向代理、静态资源服务的前端服务器。

    • Free Nginx - 社区驱动的开源分支。
    • OpenResty - 算是 Nginx + Lua 的开发平台，提供了很多优质的 Nginx 插件。
    • Kong - 现代 API 服务平台，提供诸如 API 网关等功能，其本身也利用了 OpenResty，是一个 Lua 模块 Nginx 插件。
    • ngxtop - Python 编写的工具，实时分析 nginx 的运行指标。
    • rhit - nginx 日志浏览器，指标分析工具。
    • NGINX Config - Nginx 配置在线生成工具，非官方。
    • Nginx Proxy Manager - nginx 代理服务管理器，提供 UI 交互界面，开源软件。

  • HAProxy - 可靠、高性能的负载均衡器。

  • Varnish HTTP Cache - 专用 HTTP 缓存服务器。

  • Apache - 世界上最著名的 Web 静态服务器。

  • Tomcat - Java Web 开发中的 Servlet 容器。

  • PostgREST - 可利用 PostgreSQL 构建 RESTful API 的 web 服务器。

数据库​

  • PostgreSQL - 世界上最受欢迎的开源关系数据库，开源。
    • pglite - 轻量级 Postgre，可在浏览器、Node.js 环境中使用，非 PostgreSQL 官方发布。
  • RethinkDB - 构建实时 Web 应用的数据库，开源。
  • SQLite
    • sqlite-viewer - 在线浏览器工具。
    • libSQL - 开源社区版本，非 SQLite 官方发布。

数据库管理系统​

Database Management System.

  • DBeaver Community - 图形化数据库管理工具软件，多平台支持，开源免费。
  • Falcon - SQL 客户端，支持数据可视化，多平台支持，开源免费。
  • SQLPad - Web SQL 客户端，支持数据可视化，开源软件。
