工具​

Tools.

  • Gradio - 快速构建机器学习应用。
  • Streamlit - 快速构建数据应用。
  • LocalAI - 一种本地的离线 OpenAI REST API 替代品。
  • jsonformer - 让大模型输出 JSON 结构化的数据格式。

JavaScript​

  • 模型加载（Models）

    • Transformers.js

  • 光学字符识别（OCR）

    • tesseract.js - 支持 100 种语言的图像文本识别。

  • 自然语言处理（NLP）

    • compromise

  • 数据转换（Data conversion）

    • Markdowner - 将网页转换为适合大语言模型处理的 Markdown 格式。

  • OpenLLM - 大模型的微调、部署、监控等。

开发框架​

Development Framework.

  • Common

    • LangChain - 开发基于 LLMs 的应用程序。
    • Haystack - 端到端的 NLP 框架，可构建基于 LLMs、Transformer 模型等的 NLP 应用，例如智能文档检索系统。

  • RAG

    • LlamaIndex - 数据框架，供 LLM 应用程序摄取、构建和访问私有或特定领域的数据。
    • embedchain - 构建 RAG 应用的框架。
    • PrivateGPT - 构建私有 RAG 应用的 SDK。

  • Chatbot

    • Every front-end GUI client for ChatGPT API
    • Lobe Chat
    • Chatbot UI
    • nlux
    • assistant-ui

  • Low Code

    • flowise - 开源 UI 可视化工具，用于构建定制的 LLM 编排流程和 AI 代理。

  • Agent

    • @huggingface/agents
    • AutoGen
    • SuperAGI
    • Langroid

  • Others

    • Vercel AI SDK - 开发对话流应用的 SDK。
    • LMQL - LLMs 编程语言。
